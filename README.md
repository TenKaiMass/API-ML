# API-ML

Projet algo qui class un tweet ou texte dans une class (hate, offensive, chill)

## Utilisation

- compiler le fichier `app.py`
- allez à l'adresse puis entrer un texte
- enfin clicker sur **submit** (L'algo vous donnera ensuite ça reponse)

> Le fichier algo.py est lui celui qui cree l'algo et l'entrainer, il est ensuite sauvegarder dans le fichier **algo.joblib** et recup dans app.py
